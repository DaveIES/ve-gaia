Auto-generate from Starter Geometry

The Create From Starter Geometry sub-section can be used to create a new project that is either blank (i.e. no geometry) or containing pre-defined default building geometry. This offers a fast and intuitive way of creating early-stage geometry for projects with minimal pre-required configuration.

See it in action

Create from BIM

The Create From BIM File sub-section can be used to create a new project that includes geometry and/or data from a Building Information Model (BIM) file in gbXML, IFC or veXML format.

Drawing in the VE

Spaces are created using the facilities Draw extruded shape, Draw prism, Draw pyramid, Draw sphere, Draw hemisphere and Draw cylinder. These facilities share a Shape Settings dialog which allows you to specify various attributes of the space.
